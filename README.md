# 🎮 Novel Creator (小说闯关创作工具)

这是一个基于 web 的小说闯关创作工具。

## 🚀 快速开始

1. **打开 `index.html`**（无需安装，浏览器直接打开即可）
2. **新建作品** 或 选择已有作品
3. 编写小说内容，体验闯关式创作乐趣

## ✅ 核心功能

### 🎯 排行榜与关卡系统
- 14 个关卡，每个关卡都有独特的挑战目标
- 自动计算和显示闯关进度
- 实时字数统计和排行榜

### 📊 游戏化闯关
- 按字数完成每关的创作目标
- 每完成一关，解锁新的冒险关卡
- 实时进度条和成就感

### 🌟 品质把关
- **自动润色**：AI 智能润色，保留人文特色
- **除 AI 味**：AI 生成但通过朱雀检测
- **多引擎**：支持多种 AI 供应商

### 🔧 高级工具
- **作品成品展示页 (12.6)**：管理和展示所有作品
- **公众号格式复制 (12.7)**：一键复制到公众号格式
- **支付章节标注 (12.11)**：付费章节管理

## 📁 项目结构

- `app.js` — 核心脚本（游戏逻辑与 AI 接口）
- `index.html` — Web 应用界面
- `docs/` — 设计文档 (DESIGN.md) 和变更日志 (CHANGELOG.md)
- `scripts/` — 临时审计工具（已迁移到独立仓库）
